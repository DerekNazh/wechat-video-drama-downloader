# 原项目编译指南

> 本文档说明如何编译 `wx_channels_download` 原项目为exe文件

---

## 📋 前提条件

### 需要安装

- **Go语言环境**：Go 1.19 或更高版本
  - 下载地址：https://golang.org/dl/
  - 安装后验证：`go version`

---

## 🚀 编译步骤

### 方式1：使用Go命令编译（推荐）

```bash
# 1. 进入原项目目录
cd wx_channels_download-main

# 2. 下载依赖
go mod download

# 3. 编译为exe（GUI子系统，无控制台窗口）
# -H windowsgui: 编译为 GUI 程序而非 Console 程序，运行时无控制台黑窗口
go build -ldflags="-s -w -H windowsgui -X main.Mode=release" -o wx_video_download.exe

# 4. 验证编译结果
视频号下载器.exe --help
```

### 方式2：使用批处理脚本（Windows）

创建 `build.bat` 文件：

```batch
@echo off
chcp 65001 >nul
echo 正在编译原项目...
cd /d "%~dp0"

REM 检查Go是否安装
go version >nul 2>&1
if errorlevel 1 (
    echo [错误] 未检测到Go环境
    echo 请先安装Go: https://golang.org/dl/
    pause
    exit /b 1
)

REM 编译（GUI子系统，运行时无控制台窗口）
echo 正在编译...
go build -ldflags="-s -w -H windowsgui -X main.Mode=release" -o wx_video_download.exe

if exist "视频号下载器.exe" (
    echo [成功] 编译完成：视频号下载器.exe
    
    REM 显示文件大小
    for %%F in ("视频号下载器.exe") do (
        set size=%%~zF
        set /a sizeMB=!size! / 1048576
        echo 文件大小：!sizeMB! MB
    )
) else (
    echo [失败] 编译失败，请检查错误信息
)

pause
```

## 📦 编译选项

### 优化编译（减小体积）

```bash
# 去除调试信息，减小体积
# -H windowsgui: GUI 子系统，无控制台窗口（解决打包后窗口闪烁问题）
go build -ldflags="-s -w -H windowsgui -X main.Mode=release" -o wx_video_download.exe
```

### 跨平台编译

```bash
# 编译为Windows 64位（GUI子系统）
set GOOS=windows
set GOARCH=amd64
go build -ldflags="-s -w -H windowsgui -X main.Mode=release" -o wx_video_download.exe

# 编译为Linux
set GOOS=linux
set GOARCH=amd64
go build -o 视频号下载器

# 编译为macOS
set GOOS=darwin
set GOARCH=amd64
go build -o 视频号下载器
```

## 🔧 常见问题

### 问题1：找不到go命令

**解决方案**：
1. 确认Go已安装：https://golang.org/dl/
2. 添加Go到系统PATH环境变量
3. 重启命令行窗口

### 问题2：编译失败，缺少依赖

**解决方案**：
```bash
# 清理并重新下载依赖
go clean -modcache
go mod download
```

### 问题3：编译后exe无法运行

**解决方案**：
1. 检查是否被杀毒软件拦截
2. 以管理员权限运行
3. 检查端口2022是否被占用
