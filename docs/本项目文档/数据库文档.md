# 数据库文档

## 概述

本项目使用 SQLite 数据库进行数据存储，采用分层架构设计。

---

## 数据库路径

| 环境 | 路径 |
|------|------|
| 生产环境 | `app_root/data/app.db` |
| 测试环境 | `tests/data/test.db` |

**代码引用：**
```python
from core.utils.database import db  # 单例，直接使用

# 或
from core.utils.database import Database, Author
db = Database("path/app.db")
```

## 数据模型

### Author（作者）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT | 主键，随机唯一字符串 |
| source_author_id | TEXT | 视频号原始 ID（username），UNIQUE |
| name | TEXT | 作者名称 / nickname |
| tag | TEXT | 标签 |
| bio | TEXT | 简介 / signature |
| avatar_url | TEXT | 头像 URL / headUrl |
| cover_img_url | TEXT | 封面图 URL / coverImgUrl |
| created_at | TEXT | 创建时间 |
| updated_at | TEXT | 更新时间 |
| latest_publish_date | TEXT | 最新视频发布时间（监控基准） |

### AuthorVideo（作者视频）

| 字段 | 类型 | 说明 |
|------|------|------|
| video_id | TEXT | 主键，视频 ID |
| author_id | TEXT | 外键，关联 authors.id |
| title | TEXT | 标题 / objectDesc.description |
| object_nonce_id | TEXT | 视频 Nonce ID |
| url | TEXT | 视频直链 / media[].url + urlToken |
| spec | TEXT | 视频规格 / media[].spec |
| file_size | INTEGER | 文件大小（字节） |
| cover_url | TEXT | 封面图 URL / media[].coverUrl |
| decode_key | INTEGER | 解密密钥 / media[].decodeKey |
| author_avatar | TEXT | 作者头像 / contact.headUrl |
| duration | INTEGER | 时长（秒），media[].videoPlayLen 毫秒转秒 |
| create_time | TEXT | 发布时间，createtime 时间戳转字符串 |
| is_downloaded | INTEGER | 是否已下载 (0/1) |
| download_path | TEXT | 下载路径 |

### DownloadTask（下载任务）

| 字段 | 类型 | 说明 |
|------|------|------|
| task_id | TEXT | 主键，任务 ID（对应 Go 后端 id） |
| video_id | TEXT | 视频 ID |
| url | TEXT | 视频 URL |
| title | TEXT | 标题 |
| filename | TEXT | 文件名（不含后缀） |
| spec | TEXT | 视频规格 |
| suffix | TEXT | 文件后缀 |
| key | INTEGER | 解密密钥 |
| status | TEXT | 状态：pending/running/completed/failed |
| progress | INTEGER | 进度百分比 |
| downloaded | INTEGER | 已下载字节 |
| total_size | INTEGER | 总大小 |
| speed | INTEGER | 下载速度（字节/秒） |
| error_msg | TEXT | 错误信息 |
| created_at | TEXT | 创建时间 |
| updated_at | TEXT | 更新时间 |
| completed_at | TEXT | 完成时间 |

## 表结构

### authors（作者表）

```sql
CREATE TABLE authors (
    id TEXT PRIMARY KEY,
    source_author_id TEXT UNIQUE,
    name TEXT NOT NULL,
    tag TEXT,
    bio TEXT,
    avatar_url TEXT,
    cover_img_url TEXT,
    created_at TEXT NOT NULL,
    updated_at TEXT NOT NULL,
    latest_publish_date TEXT
);
```

### author_videos（作者视频表）

```sql
CREATE TABLE author_videos (
    video_id TEXT PRIMARY KEY,
    author_id TEXT NOT NULL,
    title TEXT NOT NULL,
    object_nonce_id TEXT,
    url TEXT,
    spec TEXT,
    file_size INTEGER DEFAULT 0,
    cover_url TEXT,
    decode_key INTEGER DEFAULT 0,
    author_avatar TEXT,
    duration INTEGER DEFAULT 0,
    create_time TEXT,
    is_downloaded INTEGER DEFAULT 0,
    download_path TEXT,
    FOREIGN KEY (author_id) REFERENCES authors(id) ON DELETE CASCADE
);

CREATE INDEX idx_author_videos_author_id ON author_videos(author_id);
```

### download_tasks（下载任务表）

```sql
CREATE TABLE download_tasks (
    task_id TEXT PRIMARY KEY,
    video_id TEXT NOT NULL,
    url TEXT NOT NULL,
    title TEXT NOT NULL,
    filename TEXT NOT NULL,
    spec TEXT NOT NULL,
    suffix TEXT NOT NULL,
    key INTEGER DEFAULT 0,
    status TEXT DEFAULT 'pending',
    progress INTEGER DEFAULT 0,
    downloaded INTEGER DEFAULT 0,
    total_size INTEGER DEFAULT 0,
    speed INTEGER DEFAULT 0,
    error_msg TEXT,
    created_at TEXT NOT NULL,
    updated_at TEXT NOT NULL,
    completed_at TEXT
);

CREATE INDEX idx_download_tasks_video_id ON download_tasks(video_id);
CREATE INDEX idx_download_tasks_status ON download_tasks(status);
```

## 接口使用

### 单例模式（推荐）

```python
from core.utils.database import db

# 直接使用，无需实例化
authors = db.list_authors()
db.create_author(author)
db.delete_author(author_id)
```

### 手动实例化

```python
from core.utils.database import Database, Author

db = Database("path/app.db")
db.create_author(author)
```

### 获取单例实例

```python
from core.utils.database import get_database

    ...

```

## 注意事项

1. **单例使用**：生产环境建议使用 `db` 单例，避免重复创建连接
2. **线程安全**：Database 类使用 `threading.local()` 实现线程本地连接
3. **外键约束**：默认开启 `PRAGMA foreign_keys = ON`
4. **自动提交**：`with self._cursor()` 自动提交，出错自动回滚
